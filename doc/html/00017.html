Example Bluetooth Projects

The BM64_bootloader Demonstration 

The MCU is used as a bootloader to configure the EEPROM of the BM64 module on the BM64 Module Daughter Board. This can be done prior to using the BM64 module with the BM64_ble_comm application, for example to configure a unique BLE name for the module. When the application is running, the MCU appears as a virtual COM port on the PC. 

 

The BM64_ble_comm Demonstration 

The BluetoothÂ® Low Energy (BLE) capabilities of the BM64 Module Daughter Board are used to send a string of characters to a smartphone when one of the push buttons is pressed on the development board, and to receive a string of characters from the smartphone causing an LED to turn on or off on the board. It uses the I2S Driver, I2C Driver, codec driver (e.g. WM8904), I2S peripheral (e.g. SSC or I2SC), DMA and Timer.
